# Specifications

Specifications, formats, and DSLs.
While none of these are standards, some are useful.

<strong class="index">Contents</strong>

[:fontawesome-solid-link: **`See` header**](hateoas-see-header.md):
A simple `See` HTTP header to link to resources without polluting your JSON

[:fontawesome-solid-cloud: **`GET` search DSL**](http-get-search-dsl.md):
URI parameters that filter on boolean expressions, do not need percent-encoding, and simplify `GET` caching.
Includes extensions for images.

[:fontawesome-solid-pen: **Advanced BNF with regex**](advanced-bnf-with-regex.md):
A very powerful Backus–Naur form derivative with full regex, intersection, complement, inline labels/sub-rules,
and carefully named core rules.
