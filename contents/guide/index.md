# Setup guides

Tutorials and cheatsheets for setup and configuration.

<strong class="index">OS setup guides</strong>

_Use these step-by-step guides as **checklists** to avoid forgetting a step._
_They also dive into some advanced topics and recommendations._

[:fontawesome-solid-laptop-code: **Linux**](linux.md):
Firmware issues, GRUB bugs, advanced partitioning, filesystem compression, encryption,
repositories, useful packages, GNOME bugs, certbot,
CUDA, ZSH, SSH, GPG, Git, GitHub CLI, JDK, Node, Python, Rust

[:fontawesome-solid-laptop-code: **macOS**](macos.md):
Security, settings, Brew, ZSH, …, Rust

[:fontawesome-solid-laptop-code: **Windows**](windows.md):
Package managers, services config, forced updates, driver bugs, utilities,
Linux Subsystem, Visual C++, CUDA, ZSH, …, Rust

[:fontawesome-solid-laptop-code: **Git, SSH, and GPG**](git-ssh-and-gpg.md):
Common checklist for Git, GitHub, SSH, GPG, and signed commits

<strong class="index">Software installation guides and scripts</strong>

[:fontawesome-solid-code: **Arduino**](arduino.md):
Bash scripts to download and install
[arduino](https://arduino.cc)

[:fontawesome-solid-code: **ffmpeg**](ffmpeg.md):
Bash scripts to download, build, and install
[ffmpeg](https://ffmpeg.org/)

[:fontawesome-solid-code: **MariaDB**](mariadb.md):
Advanced configuration recommendations for
[MariaDB](https://mariadb.org/),
plus instructions and a script to install it without sudo.

[:fontawesome-solid-code: **Mamba and Conda**](mamba-and-conda.md):
Instructions to install
[MicroMamba](https://mamba.readthedocs.io/en/latest/installation/micromamba-installation.html)
and configure it to use
[Conda-Forge](https://conda-forge.org/).
*Note: I don’t use Conda/Mamba and generally advise against it.*
