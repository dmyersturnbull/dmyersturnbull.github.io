## Mathematical

!!! abstract "When to use these"

    Some characters might be used in non-mathematical contexts, such as in _(−1.6 ±0.2) µm (SE)_.
    Of course, LaTeX and MathML are always better for typesetting math.
    But in (say) source code comments, using the correct Unicode characters can improve readability.
    For example, this is quite readable:

    ```python
    # For any nonzero input vector p, p×Mₜ ∝ M₀ (for iteration t) is approximately our initial M.
    # That is, formally: ∀p≠0 ∈ ℝ³, ∃λ s.t. λp×Mₜ ≈ M₀
    # So, we first confirm that p is not the zero vector, then determine the scalar λ/lambda_.
    if np.isclose(vector, np.zeros(3), rtol=1E-7): ...
    ```
